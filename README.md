# Sistema de Pedidos Acadêmicos

Sistema para registrar pedidos e entregas de itens acadêmicos
(uniformes, mochilas, materiais básicos).

## Stack

- API: Node.js + Express (TypeScript)
- Frontend: Next.js (TypeScript)
- Banco: PostgreSQL + Prisma ORM

## Pastas

- `api/` → backend (API)
- `web/` → frontend (Next)
- `prisma/` → modelos de banco (Prisma)
